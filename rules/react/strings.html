Strings

Quotes

Always use double quotes ("") for JSX attributes. Regular HTML attributes also typically use double quotes instead of single, so JSX attributes mirror this convention.

ESLint: jsx-quotes

Examples

⇣ Incorrect code for this rule:

<Snow season='winter' />

⇡ Correct code for this rule:

<Snow season="winter" />
